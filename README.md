# Privacy Policy – Anoma

Last updated: January 2026

## 1. Introduction
Anoma is a mobile application that allows users to create an account using a pseudonym and a PIN code. We respect your privacy and are committed to protecting your personal data.

## 2. Data We Collect
We collect only the minimum data required for the application to function:
- Pseudonym chosen by the user
- PIN code (stored securely)
- Device identifier (used for security purposes)

No email address, real name, phone number, or location data is collected.

## 3. How We Use the Data
The collected data is used only to:
- Authenticate users
- Secure user accounts
- Allow account recovery on the same device

## 4. Data Sharing
We do not sell, share, or transfer your data to third parties.

## 5. Data Storage
Data is stored securely on our servers and is not shared externally.

## 6. User Rights
You can change your pseudonym or PIN at any time from within the application.

## 7. Children’s Privacy
Anoma is not intended for children under the age of 13.

## 8. Contact
If you have any questions about this Privacy Policy, you can contact us at:
support@anoma.app
