Delete your Anoma account

Anoma allows users to request deletion of their account and associated data at any time.

How to request deletion

To delete your account, please send an email to:

contact.anoma.app@gmail.com

Include the following information:

  • Your Anoma username (pseudo)
  • The subject: Account Deletion Request

Data deleted

When your request is processed, the following data will be permanently deleted:

  • User pseudonym
  • PIN code
  • Associated authentication data

Requests are processed within a reasonable timeframe.
